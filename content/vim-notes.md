> __[v]__ is visual/visual-block mode  
> __[i]__ is insert mode

# Spelling
- `z=` enable spelling
- `zg` add word to dictionary
- `zw` remove word from dictionary

# Formatting
- __[v]__ `gq` wrap selected text based on `vim.opt.textwidth`
